Quick start
-----------
1) Unzip `myco-site.zip`.
2) Edit the HTML files to add your content (no build step needed).
3) Deploy to GitHub Pages, Netlify, or any static host.

Compliance & safety
-------------------
- This project is for **education only**. Do not add instructions for cultivating or using illegal substances.
- Add clear disclaimers and link only to trusted, official sources.
- Laws change frequently; verify locally.

Suggested next steps
--------------------
- Replace `hello@example.com` on /contact.html
- Update `site_url` in the `<head>` tags and `sitemap.xml`
- Add real analytics (e.g., Plausible) and a newsletter provider (e.g., Buttondown or Mailchimp)
- Swap colors and fonts to match your brand
- Create real blog posts under `/blog/`

SEO
---
- Unique titles & descriptions per page.
- Use alt text for images.
- Add canonical URLs and Open Graph images if you have them.
